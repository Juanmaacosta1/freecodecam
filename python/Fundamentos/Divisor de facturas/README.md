#  Divisor de Billetes - Bill Splitter

##  Proyecto en Python | freeCodeCamp

Este taller consiste en construir un pequeño programa que calcula cuánto debe pagar cada persona después de sumar los costos de una comida, agregar una propina y dividir el total entre un grupo de amigos.

Es un ejercicio práctico enfocado en el trabajo con números, operaciones matemáticas y actualización de variables en Python.

---

## Objetivo del proyecto

- Practicar el uso de variables numéricas
- Realizar operaciones matemáticas básicas (+, *, /)
- Usar operadores de asignación aumentada (+=)
- Aplicar la función `round()` para formatear resultados
- Simular un escenario real de división de gastos

---

##  Conceptos aplicados

- Variables tipo float
- Suma de múltiples valores
- Multiplicación para cálculo de porcentaje (propina)
- División entre varias personas
- Redondeo a dos decimales
- Impresión de resultados con `print()`

---

## Lógica del programa

1. Se definen los costos individuales:
   - Aperitivos
   - Platos principales
   - Postres
   - Bebidas

2. Se calcula el total acumulado.
3. Se calcula una propina del 25%.
4. Se suma la propina al total.
5. Se divide el total final entre el número de amigos.
6. Se redondea el resultado a dos decimales.
7. Se muestra cuánto paga cada persona.

---

##  Tecnologías utilizadas

- Python 3
- Operadores matemáticos
- Funciones integradas (`round`, `print`)

---

##  Resultado esperado

El programa muestra:

- Total acumulado
- Monto de la propina
- Total con propina
- Cuánto paga cada persona
- Valor final redondeado a dos decimales
