Cree un planificador meteorológico de viaje
Para este laboratorio, utilizará declaraciones condicionales para determinar si es posible viajar en función del clima, la distancia a recorrer y la disponibilidad de un vehículo.

Objetivo: Complete las historias de usuario a continuación y obtenga todas las pruebas para aprobar para completar el laboratorio.

Debes crear las siguientes variables:
distance_mi(un número que representa la distancia a recorrer en millas)
is_raining(un valor booleano que representa si el usuario está experimentando actualmente un clima lluvioso)
has_bike(un booleano que representa si el usuario tiene una bicicleta)
has_car(un valor booleano que representa si el usuario tiene un automóvil)
has_ride_share_app(un booleano que representa si el usuario tiene una aplicación que le permite solicitar un viaje)
Debe utilizar declaraciones condicionales para determinar si es posible realizar desplazamientos en función de los valores de estas variables.
Deberías usarlo if, elif, y else declaraciones para evaluar las categorías de distancia en orden ascendente.
Si distance_mi es un valor falso:
Deberías imprimir False.
Si la distancia es menor o igual a 1 milla:
Deberías imprimir True sólo si lo es No llueve.
De lo contrario, deberías imprimir False.
Si la distancia es mayor a 1 milla y menor o igual a 6 millas:
Deberías imprimir True sólo si la persona tiene bicicleta y No está lloviendo.
De lo contrario, deberías imprimir False.
Si la distancia es más de 6 millas:
Deberías imprimir True si la persona tiene coche o tiene una aplicación de viajes compartidos.
De lo contrario, deberías imprimir False.
